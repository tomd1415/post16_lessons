Lesson plan

Overview

Recursion is possibly the hardest topic in the course. The main point for students

at this level to absorb is that recursion works by a function calling itself.

See the file Fern.py in lesson 14 in the StudentCodeFiles folder relates to the

Example: Recursion for Fractal Fern in Section 14.2. The Curve.py in lesson 14

in the TeacherCodeFiles folder relates to the Example: Drawing a Curved Line in

Section 14.1.

Learning objectives

  • Understand the term recursion
  • Loop
  • Procedure
  • Function
  • Variable
  • Parameter

Exercises in this lesson

  • Tower of Hanoi — Understand recursion
  • Recursive Pattern — Base case, recursive step, and depth-controlled output
