Printable cards

Print and cut these cards for quick prompts or station work.

Symbol legend

  • Start/End — oval
  • Process — rectangle
  • Decision — diamond (Yes/No)
  • Input/Output — parallelogram

Everyday algorithm

  • Pick a routine (breakfast / pack bag).
  • Add 1 decision branch.
  • Show Start → steps → End.

Inputs & outputs

  • Device: camera / washing machine / microwave.
  • List 2 inputs + 2 outputs.
  • Draw IO as parallelograms.

Machine flowchart

  • Decision with Yes/No labels.
  • Check for loop exit (timeout/condition).
  • Add End node.

Debug checklist

  • Do arrows flow top→bottom?
  • Does every decision have two exits?
  • Is there an End for each path?
