Lesson plan

Overview

This lesson uses Python as a calculator. Students practice operator precedence and parentheses, then apply them while crafting expressions.

They build expressions to hit targets (Four 4's) and step through evaluation to debug and fix mistakes (Evaluate Expressions).

Learning objectives

  • Recognise and use arithmetic operators: +, -, *, /
  • Know how parentheses affect the evaluation of mathematical expressions
  • Understand and apply the precedence of operators in complex expressions
  • Understand how to use parenthesis to structure complex expressions

Exercises in this lesson

  • Four 4's — Using Python as a calculator
  • Evaluate Expressions — Apply precedence and parentheses
