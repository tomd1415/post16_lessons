Review question answers (from handbook)

1. An event is:

b. Something that happens that a program should react to.

2. The function 'randint' is found in which library?

d. The random library.

3. The string named greeting has the value 'Good'. Which command sets it to have

the value 'Good Morning'?

b. greeting = greeting + " Morning"

4. The tuple named colour has the value (0,0,255). Which command sets it to have

the value (255,0,255)?

c. colour = ( 255 , 0+0 , 255 )

5. Which of the following does not generate events in Python with Pygame?

d. Battery low.
