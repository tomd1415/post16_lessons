Lesson plan

Overview

The purpose of this lesson is to allow students to familiarise themselves with the

Python shell and gain early success in creating, running and saving a simple

program.

Note: You must have Python installed on each student computer before

beginning this lesson. Refer to Installing Python in this handbook for

instructions.

Learning objectives

  • Start and run a program
  • Enter code
  • Create and save a program
  • Open and run a program

Exercises in this lesson

  • Aliens! — Getting started with Python programs
