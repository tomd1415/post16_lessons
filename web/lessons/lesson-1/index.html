Thinking like a Coder — Lesson 1 Offline interactive resources (HTML/CSS/JS)

Teacher hub

What this pack contains
  • Interactive activities for the Lesson 1 themes: computing computational thinking decomposition pattern recognition abstraction algorithms
  • Teacher notes + printable materials (print from the browser).
  • Offline-first (no internet required once downloaded).

Quick start

  1. Unzip the folder.
  2. Open index.html in a browser (Chrome/Edge/Firefox).
  3. Project activities from Student hub or send learners student.html.
If your school blocks local JS in the browser, use a portable browser or host the folder on an internal web server.

Lesson intent (from ICDL)

Lesson 1 focuses on computational thinking: how problems can be analysed and broken into smaller chunks, and why computers are strong at tasks that can be expressed as clear, simple steps.

Teacher mode note: Toggle teacher mode to reveal suggested answers and example approaches in activities.
