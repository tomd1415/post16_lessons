This folder is an **offline-first** set of interactive resources for **Lesson 1: Thinking Like a Programmer**.

## Open it

- Open `index.html` (teacher hub) in a browser.
- Send learners to `student.html` (student hub).

## Teacher mode

- Click **Teacher mode** in the top bar, **or**
- Add `?teacher=1` to any activity URL.

Teacher mode reveals hints / answer guidance inside activities.

## Printing

- Printable cut-out cards: `teacher/print-cards.html`
- Use your browser print dialog.

## Data privacy

Progress is stored in **local browser storage** on that machine only.

## Tested

Designed for modern Chromium browsers (Chrome/Edge) and Firefox.
