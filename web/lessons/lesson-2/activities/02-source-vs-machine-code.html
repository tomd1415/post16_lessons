Teacher guidance (answer prompts)

  • Quick checks: (1) humans, (2) binary instructions, (3) compiler/assembler.
  • Task A: 1=M, 2=S, 3=S (low-level but still source), 4=S (comment, not executed).
  • Task B: missing step is compile/assemble; purpose is to translate to CPU-ready binary.
  • Task C: Expect “load 2 and 3 into registers, add, store result”.
  • Misconceptions: “Python is machine code”; “comments run”; “CPU reads .py directly”.
  • Differentiation: pair a novice with a peer coach to narrate the pipeline aloud; stretch to name real opcodes.
