Lesson plan

Overview

The final two data types, list and tuple are introduced in this lesson. The Python

'list' aggregate data type fulfils the role of an 'array' in other programming

languages. Rather than holding a single value, these data types hold multiple

items, for example, a list of names. Some basic examples illustrate their use.

Learning objectives

  • Understand aggregate data types
  • Use aggregate data in a program like: list and tuple

Exercises in this lesson

  • Reversing a list — Working with lists
