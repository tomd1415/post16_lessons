Lesson plan

Overview

Students step away from sequential statements in this lesson and explore some

basic looping structures. The arrow head example from lesson 11 is further

developed here.

See the files ArrowBig.py and ArrowSmall.py in lesson 12 in the

StudentCodeFiles folder which relate to the Example: Drawing an Arrowhead in

Section 12.4 of the student learning materials.

Learning objectives

  • Define the programming construct term loop. Outline the purpose and benefit of looping in a program
  • Recognise types of loops used for iteration: for, while, repeat
  • Understand the term infinite loop
  • Use iteration (looping) in a program like: for, while, repeat

Exercises in this lesson

  • How big can a Python integer be? — Infinite loop, Integers
  • Horizontal Arrow — Loops, Procedures, Functions and Parameters
