Step 2: Key ideas in your own words

Explain the main idea of How big can a Python integer be? in one or two sentences.

Step 3: Build the main task

Learning objective: Infinite loop, Integers

Suggested use: Student activity

Ask the students to write a while loop that doubles a number repeatedly.

How big can a Python integer get?

Python is different from many other computer languages. There isn't a

hard fixed limit to the size of an integer in Python. Instead Python can

keep doubling a number until it runs out of memory to store the number in.
