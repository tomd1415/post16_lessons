a01 - Logic in Search Terms

How this works: This activity is broken into small steps with quick checks and spaces to add your own ideas.

Step 1: Quick checks

Prompt 1: AND is true only when...

Prompt 2: OR is true when...

Prompt 3: NOT does what to a Boolean value?

Step 4: Reflection

Teacher guidance (answer prompts)

  • Quick check 1: both conditions are true
  • Quick check 2: at least one condition is true
  • Quick check 3: Flips it
  • Learning objective focus: Algorithms (from lesson 1)
  • Step 2 answers: AND = both true; OR = at least one true; NOT = exclude the term
  • Task A mapping: AND → all these words; OR → any of these words; NOT → none of these words
  • Task B examples: pumpkins -halloween; snowflakes -song -lyrics
  • Task C: expect a relevant page title/URL for each search (or a predicted title if offline) and a checked box if found
